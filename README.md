# Healthcare-Data-Analysis-Using-SQL
## Project Overview
##### This project focuses on analyzing a healthcare dataset using SQL to uncover meaningful insights about patient distribution and demographics.
##### The goal is to demonstrate strong foundational SQL skills including data aggregation, grouping, and database handling—key skills for data analyst roles.
## Objectives
##### - Create and manage a structured healthcare database
##### - Perform data analysis using SQL queries
##### - Generate insights from patient data
##### - Practice real-world data querying scenarios
## What I Did (Process)
## Database Setup
##### - Created a healthcare database in MySQL.
##### - Imported CSV datasets into four tables.
## Data Analysis Using SQL
#### *I performed different types of SQL operations:(Basic Queries)
##### - Total patients
##### - Gender-wise and city-wise patient distribution
##### - Average patient age
##### - Senior citizen count
##### - Total doctors
##### - Specialization-wise doctor count
##### - Total appointments
##### - Total hospital revenue
## Aggregation & Grouping
##### - Age-wise patient grouping
##### - Disease-wise appointment count
##### - Payment mode-wise revenue
##### - Month-wise revenue trends
##### - Doctor-wise appointment count
## JOIN Operations
##### Used:
##### - INNER JOIN
##### - LEFT JOIN
##### - Multi-table joins
##### - Performed advanced analysis such as:
##### - City-wise revenue
##### - Doctor-wise revenue
##### - Specialization-wise revenue
##### - Patient-wise total spending
##### - Top 3 doctors by revenue
##### - Disease-wise revenue
##### - Doctors without appointments
##### - Repeat vs new patients
##### - High-value patients identification
## SQL Concepts Used
##### - SELECT

##### - GROUP BY
##### - HAVING
##### - ORDER BY

##### - JOIN (INNER JOIN, LEFT JOIN)

## Project Insights
### From this analysis, I found:
##### - Which cities generate the highest hospital revenue
##### - Which doctors earn the most revenue
##### - Which specialization brings more income
##### - Most common diseases treated in the hospital
##### - Total spending of each patient
##### - Top 3 doctors based on revenue
##### - Difference between new patients and repeat patients
##### - Patients who consult multiple doctors

## Final Conclusion
##### This project helped me understand how SQL can be used to analyze real-world healthcare data.
##### By using joins, grouping, and aggregate functions, I was able to combine multiple tables and generate meaningful insights about patients, doctors, and hospital revenue.
#### The analysis shows:
##### How revenue is generated across cities and doctors
##### Patient visit patterns (new vs repeat patients)
